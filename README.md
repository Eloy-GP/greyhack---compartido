# greyhack---compartido

Este repositorio es para compartir los avances del juego.

## Objetivo

La idea de este proyecto es tener un espacio común donde subir, guardar y revisar el progreso del juego a medida que se va desarrollando.

## ¿Qué se comparte aquí?

- Avances del desarrollo
- Archivos y recursos del juego
- Cambios importantes del proyecto
- Material relacionado con el progreso del juego

## Propósito

Facilitar la colaboración y mantener organizados los avances para que todos los participantes puedan ver el estado actual del juego y continuar trabajando sobre una base compartida.
